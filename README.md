# PowerShell Attack Investigation Lab

This project demonstrates a **SOC investigation of a simulated PowerShell attack** using Splunk Enterprise on Ubuntu.  

## Features
1. Ingest PowerShell logs in Splunk  
2. Detect suspicious PowerShell executions  
3. Detect encoded commands (`-enc`)  
4. Decode malicious Base64 commands  
5. Map attacks to MITRE ATT&CK (T1059)  
6. Include screenshots for documentation  

All logs and screenshots are included in this repository.
